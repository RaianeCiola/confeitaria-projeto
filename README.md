# Sistema de Confeitaria

Projeto desenvolvido para a disciplina de **Sistemas Distribuídos** em 2024.

A aplicação consiste em uma plataforma distribuída baseada em arquitetura orientada a serviços, composta por Front-End, API (Back-End) e Banco de Dados Relacional.

## Tecnologias Utilizadas

- Python
- FastAPI
- Banco de Dados Relacional
- Front-End Web
- Documentação da API: Swagger
- Testes de API: Insomnia

## Arquitetura

O sistema foi desenvolvido seguindo o modelo de arquitetura distribuída:

Front-End → API (Back-End) → Banco de Dados

## Estrutura do Projeto

api_confeitaria → código da API  
confeitaria_front → interface da aplicação  
database → script SQL para criação do banco de dados

## Objetivo

Aplicar conceitos de **Sistemas Distribuídos**, incluindo comunicação via API REST, integração entre serviços e persistência de dados em banco relacional.
